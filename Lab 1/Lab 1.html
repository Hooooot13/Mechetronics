Lab introduction

The Mega 2560 is an EVB (EValuation Board) created by the Aruino project built around Atmel's ATmega250 microcontroller. The Mega2560 is capable of handling advanced projects. We will use the capabilities of the board throughout the semester in various applications. The purpose of this introductory lab is to ensure proper set up of the programming environment, Arduino Software (IDE). Your technical objective for the lab is to create an electronic recipe for one entree of your choice. You recipe should provide the recipe (ingredients and instructions) timed appropriately so that one can follow the recipe to make the dish. You should test the result of this exercise on a friend and report on the outcome.

Download the software

The Arduino MEGA board requires ARDUINO 1.6.7 or newer
Download ARDUINO 1.6.7 or newer, this enables you to write and upload code to the Arduino board. It can be installed on Windows or Mac operating systems.

  • Arduino MEGA 2560

  1. To download, go to the Arduino.cc website https://www.arduino.cc/en/Main/Software
  2. After selecting the operating system, follow the instructions to download the latest Arduino environment. Install the software after it is downloaded.
  3. If necessary, follow the instructions on how to install the drivers.

Hardware Setup

This section will provide instruction on the first time you use the Arduino Hardware. You will need the ATmega 2560 on and evb (the Arduino mega) and an lcd display. 

  1. Assemble basic components

  • Arduino MEGA 2560

  1. plug the USB cable into your computer and board.
  2. Launch the Arduino software and select the appropriate board from Tools > Board > Arduino Mega or Mega 2560.
  3. Select the serial port by clicking Port in the tools menu Tools > Port > COM3. MAKE SURE YOU CLICK ON THE PORT. Typically the default is COM3.
  4. if you are unsure which port to use:
  • Click "Start"
  • Right click on "Computer"
  • Select "Manage"
  • Select "Device Manager"
  • Expand the "Ports (Com & LPT)" section
  • Note the number of the COM port (e.g. COM7) used by the USB
  • Ensure that the COM port number is between COM1 and COM8.
  • If not, reassign the USB to a COM port in that range.
Right Click, on the COM PORT and go toProperties.
Click on Port Settings and then click on Advancebutton.
Chose a port number between 1 to 8
In the "Action" menu, choose"Scan for hardware changes"

To run an existing program from Arduino Examples

1. Launch the Arduino application.
2. Go to File >Examples > 01.Basics > Blink. This will open a pre-written program that controls the LED on pin 13. 
3. You can alter the on and off times by changing the value in delay(1000). delay(1000) calls the function delay and sends it a delay time in milliseconds before moving to the next command.

4. press the arrow button to upload your program and check to see that theLED on and off times are as you specified in the program.

Writing your First Program

Introducion

This section will walk you through the process of creating your first program with the ARDUINO 1.6.7 programming toolkit. In this exercise you will create a simple program to print "Hello World!" to the serial screen.

Procedure 1. Launch ARDUINO 1.6.7
2. This gives a blank program to you. It contains two parts, Setup{ } and Loop{ }. Think of setup as everything that would go before main and Loop as Main.
3. Create setup part of code. enter the following code to initialize a printer to the serial monitor. 

 void setup() { // put your setup code here, to run once:
 Serial.begin(9600);       // Initialize the Serial
   while (! Serial);       // Wait for Serial to be ready
   Serial.println("Serial is ready"); // print to the Serial port
  }

4. Go to Loop and create the main part of code. Enter the following code: 

 void loop() { // put your loopcode here, to run repeatedly:
   Serial.println("Hello World!"); // print to the Serial port
   delay(1000);  // delay 1000 ms
   Serial.println("goodbye World!); // print to the serial port
   delay(1000);  // delay 1000 ms
 }

5. upload the file by pressing the right arrow at the top of the ide
6. If any errors occur, check your board selection and port. Then go to the online debugger
7. To see the output, open the serial monitor. Go to Tools > Serial Monitor
8. You should see output as programmed
