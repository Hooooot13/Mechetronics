********* PIN ORDER *********

-Servos
  -Servo 1 (Front Door)
    -Ground  -> Ground
    -Power   -> 5v Power
    -Signal  -> Pin:11 (PB5)

  -Servo 2
    -Ground  -> Ground
    -Power   -> 5v Power
    -Signal  -> Pin:13 (PB7)

-Button Pad
  -Button Zero  -> Pin:49 (PA0)
  -Button One   -> Pin:48 (PA1)
  -Button Two   -> Pin:47 (PA2)
  -Button Tree  -> Pin:46 (PA3)

-Range Finder
-Ground  -> Ground
-Power   -> 5v Power
-Signal  -> Pin:A0 (PF0)
